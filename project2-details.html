Nafsity Website

Project Overview

Nafsity is a mental wellness web application focused on improving users' psychological well-being through interactive self-assessment and personalized mood enhancement advice. The platform offers users an accessible and engaging experience to evaluate their mental state and receive tailored tips for emotional balance.

The site uses browser-based quizzes combined with logic-driven feedback to help users reflect on their psychological status. With a clean and fully responsive design, it supports Arabic language content and smooth user interaction.

Technologies Used

HTML5
CSS3
JavaScript
AOS Library
Font Awesome

Project Duration

7 days (March 2025)

Role

Frontend Developer

Key Features

Arabic-language mental health quiz
Real-time psychological assessment results
Mood improvement suggestions based on quiz answers
Smooth scrolling animations using AOS
Fully responsive design across all devices
Visually engaging and user-friendly interface
